# Weather App

A simple and responsive Weather App built using HTML, CSS, and Vanilla JavaScript. The application allows users to search for any city and view its current weather information using the Open-Meteo Geocoding and Weather APIs.

## Features

* Search weather by city name
* Display current temperature
* Display weather condition
* Display humidity
* Display wind speed
* Press Enter to search
* Loading indicator while fetching data
* Handles invalid city names gracefully
* Responsive and clean user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Open-Meteo Geocoding API
* Open-Meteo Weather API

## How It Works

1. User enters a city name.
2. The Geocoding API converts the city name into latitude and longitude coordinates.
3. The Weather API uses those coordinates to fetch current weather information.
4. Weather details are displayed on the screen.

## API Endpoints

### Geocoding API

Converts city names into coordinates.

https://geocoding-api.open-meteo.com/

### Weather API

Provides current weather data.

https://open-meteo.com/
